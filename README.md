# miniMarkdown

## Getting started

Use function `minimarkdown()`.

```
$text = '# H1
## H2
### H3
_Italic_
**Bold**
`Code`
++Positive++
--Negative--
![Alt](https://www.google.com/intl/en_ALL/images/logo.gif)
[Google](https://www.google.com)
[email](mailto:mail@mail.com)';

echo minimarkdown($text);
```
